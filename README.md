# vue2原型项目

## Issues
使用的时候遇到任何问题或有好的建议，请点击进入[issue](https://github.com/agile-contrib/vue2/issues)，欢迎参与维护！

## How to use?

开发的时候运行下列命令：

```
npm run dev
```

开发完毕需要发布的时候，请运行下列命令：

```
npm run build
```
